# Home Assistant Automation Blueprints

Ready-to-use automation blueprints for Home Assistant.

## IKEA Bilresa Unified Controller

Complete control for IKEA Bilresa (RODRET E2201) scroll wheel with full 3 layers Ã— 3 buttons support.

### Features
- **3-layer support**: Each wheel position (left/center/right) acts as a separate layer
- **Multi-press detection**: Single, double, and triple press actions per layer
- **Hold actions**: Long press support for all buttons
- **Brightness control**: Scroll wheel adjusts light brightness with configurable steps
- **Scene activation**: Assign scenes to button presses
- **Cover control**: Control blinds/covers with scroll wheel

## ESPHome Auto-Updater

Automatically update ESPHome devices when new firmware versions are available.

### Features
- **Automatic updates**: Detects and installs ESPHome updates without manual intervention
- **Update windows**: Schedule updates during specific time periods (e.g., night hours)
- **Notifications**: Get notified when updates start, complete, or fail
- **Selective updates**: Choose which devices to auto-update
- **Retry logic**: Handles temporary failures gracefully
