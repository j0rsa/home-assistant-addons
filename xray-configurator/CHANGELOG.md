# Changelog

## 1.2.0

- Add optional DNS configuration with customizable DNS servers (defaults to 1.1.1.1 and 8.8.8.8)
- Add option to resolve server hostname to IP address via DNS-over-HTTPS
- Fix routing rules to only include inbound tags for enabled proxy types
- Add inbound-to-outbound routing rule for Shadowsocks configurations

## 1.1.3

- Fix theme detection and copy buttons

## 1.1.2

- Initial release with VLESS and Shadowsocks link conversion
- HTTP and SOCKS proxy inbound configuration
- SOCKS authentication support with dynamic user management
- JSON and Base64 output with copy-to-clipboard
- Light and dark theme support
